# Simple Calculator

A simple JavaScript calculator with basic operations: addition (+), subtraction (-), multiplication (*), and division (/).

## Features
- Simple and lightweight
- Supports +, -, *, / operations
- Clear (C) button to reset

## How to Use
1. Open `calculator.html` in any web browser
2. Click the buttons to perform calculations

## Tech Stack
- HTML
- CSS
- JavaScript
